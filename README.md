## 开发

### Node.js

请使用 Node 14 版本
### Yarn

安装 Yarn：

`npm install -g yarn`

安装 Yrm：

`npm install -g yrm`

切换成淘宝源加速安装：

`yrm use taobao`

安装依赖：

`yarn`

### 启动

`yarn start`
## 发布

`yarn run createDaily`

`yarn run daily`

`yarn run publish`

切换到 gh-pages 分支，修改 index.html 的 cdn 前缀，commit & push 即可
